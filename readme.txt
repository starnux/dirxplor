DirXplor Beta 1.5 by Na-Prod
------------------------------------------------

Explorateur avec les fonctions de Dirent.h de Turbo C++ 3.
Permet de voir les lecteurs et les dossiers !
Affichage super amelioré et convivial !
On peu maintenant afficher des Bmp de toute tailles N&B dans DirXPlor !
La lecture des Fichiers CSV est Totale, il permet de les lire a haute vitesse !
Prenez Flash100 version c++ derniere version pour faire
des dossiers dans des lecteurs ou uploadez le lecteur
inclus dans le Zip.
